# MLC Forecast

## Forecast for MLC price

### Download and active enviroment

Config virtualenv:

```bash
$ git clone https://github.com/eadomenech/mlcForecast.git
$ python3 -m venv env
$ source env/bin/activate
```

### For local development

Install requirements:

```bash
$ cd mlcForecast/
$ pip install -r requirements.txt
```
